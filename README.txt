Documentation for the project html package.

Inside the folder Project04_AMedeiros.html exists the following files:
 - index.html (Containts web page content)
 - script.js	(Contains scripts to run the interactive function of the web page) 
 - style.css	(CSS style sheet for 'light mode')
 - README.txt	(Documentation)

Each file supports the functionality built into the index html file. 

To use this webpage, open index.html in any modern web browser (tested in Firefox, Edge, Opera GX)

Image content is linked directly to the source web page and is not contained locally. 

Chart is rendered courtesy of Chart.js API

CoinCap 2.0 RESTful API interactivity is linked directly to the https service. 

This webpage will allow the user two optios:
    1.) Search and chart a cryptocurrency of their choice. Timescale of year(default), month, or day can be selected. Chart is redrawn with each new search and selection.
    2.) Display the top ranked crpytocurrey according to CoinCap.
